## 使用方法
使用命令格式：`/aimg <提示词>`

## 配置说明
- API地址：文生图服务器地址
- API密钥：访问密钥
- 模型：使用的AI模型
- 分辨率：生成图片的尺寸
- 推理步数：生成质量控制
- 种子ID：控制生成结果的随机性
